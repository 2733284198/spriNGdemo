Java EE 8 and Angular 4 (cli) demo

In this demo we show some of the new features of Java EE 8 and how these can be integrated with a JavaScript frontend.

Why Angular as frontend and not a Java view framework?

There are other demos that cover JSF [link]. Our goal is to show how Java EE is a perfect and modern companion to modern JavaScript frameworks.

Java EE 8 status

Java EE 8 is currently under heavy review and development. The release of the final specification is annouced for July 2017. This demo is based on alpha releases of the Application servers and single component. The goal is only to show the new features of the platform and not to build a production ready application.
