Spring Boot 2.0 and Angular 4 (cli) demo

In this demo we show some of the new features of Spring Boot 2.0 and some JSR part of Java EE 8. We integrate these new feature a JavaScript / Reactive frontend.

Why this demo?

The original idea was to cover the new features of Java EE 8. You can visit the sister demo here javademo.io.

Spring and Java EE are great platforms to build software and Java benefits greatly by the contribution of these 2 communities. For this reason I'm doing the effort to build a similar website for Spring.
The content of the two demos will be similar but different like the 2 platforms ;-) The Spring version will be a bit more customized and personally branded than the Java EE version.

The Java EE demo has want to be a present for the Java EE (JCP) community and a neutral demo.

You can test yourself the two platform and contribute to the demo. You are invited to actively contribute to your favourite Java project. It's a great personal experience and an important contribution to the future of Java.

Status and future development of this app

Currently this demo is migrated from Java EE, there are still a lot of small problems to fix, be patient.
The demo will be regularly updated with the new features offered by Spring and Angular. Check GitHub (or my blog or twitter) for the updates.
